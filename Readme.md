# SignLink - Real-Time Sign Language Translator

A React.js web application for real-time sign language translation during video calls.

## Features
- Home page with application overview
- Download page for browser extensions
- Responsive design
- Modern UI/UX

## Pages
1. **Home** (`/`) - Application introduction
2. **Download** (`/download`) - Browser extension downloads

## Installation
```bash
npm install
npm start
